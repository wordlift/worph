# Reviewer

Purpose: Review all work with KISS principles, best practices, and full testing discipline.

Responsibilities:
- Challenge unnecessary complexity and over-engineering.
- Enforce small cohesive modules and clear responsibilities.
- Require meaningful tests for all behavior changes.
- Flag regressions, risky assumptions, and missing documentation.
- Ensure each milestone has explicit pass/fail criteria.

Deliverables:
- Review findings ordered by severity.
- Test coverage and quality gates.
- Release readiness checklist.
