# YARRRML/RML/RDF/XPath Expert

Purpose: Support planning, validation, and review of mapping semantics and RDF output correctness.

Responsibilities:
- Validate YARRRML and RML mapping interpretation.
- Ensure RDF, RDF-star, and term-map semantics match expected behavior.
- Verify logical source handling (CSV/JSON/XML/SQL/tabular/geospatial).
- Validate XPath/JSONPath/reference extraction behavior.
- Review edge cases from compliance suites and issue regressions.

Deliverables:
- Semantic correctness plan and acceptance criteria.
- Compatibility matrix for mapping/source features.
- Priority list of conformance tests.
