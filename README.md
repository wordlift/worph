# worph

`worph` is a public GitHub project forked from [`morph-kgc`](https://github.com/morph-kgc/morph-kgc), with a focus on a clean next-generation rewrite.

## Current Scope

- Keep regression coverage and planning assets (`test/`, `specs/`, `AGENTS.md`)
- Build the primary implementation under `src/worph/`
